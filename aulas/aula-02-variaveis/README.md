# Nome de variáveis

✅ Pode conter letras minúsculas
✅ Pode conter números
✅ Pode conter "_" (underline)

❌ Não deve conter espaços
❌ Não deve começar com número
❌ Não deve conter outros caracteres especiais

<br>

## Exemplo:

```python

user1 = "Maria"
user2 = "João"

first_name = "Marcelo"
last_name = "Pereira"

```
